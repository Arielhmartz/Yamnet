# Yamnet
modelo para analisis de sonidos basado en google 
1. El Repositorio Maestro (TensorFlow Models)
YAMNet no tiene un "GitHub propio" como un proyecto independiente, sino que vive dentro del ecosistema oficial de modelos de Google. Lo encontrarás aquí:

Ruta en GitHub: tensorflow/models/tree/master/research/audioset/yamnet

Qué hay ahí: El archivo inference.py (el corazón para procesar audios), el archivo params.py y el mapeo de clases (yamnet_class_map.csv) donde verás la etiqueta de la motosierra.

2. La vía rápida: Kaggle Models (Sustituto de TFHub)
Google ha movido la descarga directa de los archivos .tflite (para móviles o dispositivos pequeños) y los modelos pesados a Kaggle.

Busca en Google: "Kaggle YAMNet model".

Desde ahí puedes descargar el modelo ya entrenado sin tener que compilar nada de código complejo.
